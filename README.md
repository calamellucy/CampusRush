# Campus Rush v2.0.0

### 프로젝트 설명

지각 위기에 처한 대학생이 지각하지 않기 위해 달리는 러너 게임.
Unity 2D 기반 팀프로젝트로, 다양한 아이템과 이벤트 시스템이 추가되어 더욱 역동적인 플레이가 가능합니다.

---

### v2.0.0 주요 업데이트 상세
#### 1. 아이템 시스템 고도화
- 신규 아이템 추가: 학점 아이템(A, B, C, D, F) 및 체력 회복 하트 아이템 추가.

- 점수/체력 로직:
	- 학점별 점수 차등 적용 (A:+500, B:+400, C:+300, D:+200, F:-300).
	- 체력 아이템 획득 시 HP 1칸 회복 (최대 3칸 제한).
	- Item.cs 최적화: 점수 및 체력 변화가 있는 아이템만 선택적 처리.

- 확률 조정: 아이템별 등장 가중치 재설정 (학점/체력/커피).

#### 2. 이벤트 시스템 (Buff/Debuff)
- 이벤트 효과 구현:
	- 교수님 이벤트: 아이템 점수 및 시간당 점수 2배 (마이너스 점수는 0점 처리).
	- 연애 이벤트: 아이템 점수 2배 적용.
	- 사이비 이벤트: 모든 점수 및 시간당 점수 0.5배 적용.
  
- 시스템 개선: ScoreManager.cs 및 BuffManager.cs 연동을 통한 이벤트 실시간 적용 및 초기화 로직 구현.

#### 3. 플레이어 애니메이션 개선
- 에셋 교체: 달리기 및 숙이기 애니메이션 최신화.
- 점프 메커니즘 고도화: Player_JumpUp, Player_JumpDown 클립 추가.
- 상태 제어: IsGrounded 및 YVelocity 파라미터를 사용하여 상승/낙하 상태 전환을 자연스럽게 개선.

#### 4. 에셋 리뉴얼 및 시각적 개선
- 바닥 에셋 전면 교체 및 적용
- 자동차 및 장애물 에셋 최적화 적용
- 커피 아이템 에셋 디자인 및 적용
- 이벤트 NPC 에셋 적용 및 플레이어 Collider 세밀화 수정
- 애니메이션: 달리기/숙이기 애니메이션 최신화, 점프 상승/낙하 상태 애니메이션(JumpUp, JumpDown) 추가.


### 주요 기능 (요약)

- 플레이어: 숙이기, 점프, 2단 점프, 애니메이션 상태 제어

- 시스템: HP 시스템(최대 3), 이벤트 시스템, 점수 계산 시스템

- 오브젝트: 학점 아이템, 회복 아이템, 장애물, 배경 요소

- Git
- GitHub Desktop
- Windows 또는 macOS

- Unity 2D 프로젝트 실행이 가능한 PC

### 설치 및 설정 방법

1. GitHub 저장소를 클론

```
git clone 저장소_URL
```

2. 프로젝트에 맞는 Unity 버전으로 프로젝트 열기

3. Assets/Scenes 폴더에서 StartScene을 열고 실행

### 기여 방법

브랜치 전략: Git Flow

- main: 최종 배포 브랜치
- develop: 개발 통합 브랜치
- feature/\*: 기능 개발 브랜치
- release-\*: 릴리즈 준비 및 QA 브랜치
- fix/\*: 버그 수정 브랜치

작업 방식

- Issue 탭에 이슈 등록
- develop 브랜치에서 feature 브랜치 생성
- 기능 구현 후 Pull Request 생성
- 코드 확인 및 테스트 후 develop 브랜치에 병합
- 병합 후 해당 feature 브랜치 삭제
